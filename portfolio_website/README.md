## Overview
The central hub for the Data Science, Engineering, and AI laboratory. This Next.js application serves as the portal to various interactive experiments and projects, featuring a modern, responsive design that matches the aesthetic of the Control Engineering and Math labs.

## Features
- **Modern UI**: Built with Next.js 15, React 19, and Tailwind CSS v4.
- **Responsive Design**: Optimized for mobile and desktop devices.
- **Project Showcase**: Grid layout displaying available laboratory modules with status chips.
- **Dark Mode**: Automatic theme switching based on system preferences.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **Font**: Space Grotesk & Manrope (via `next/font`)
- **Deployment**: Vercel

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
