# Applied Mathematics 2: Differential Equations

## Overview
Numerical solutions to ordinary and partial differential equations using various discretization schemes and time-stepping methods.

## Features
- **ODE Solvers**: Euler, RK4, adaptive step size
- **Boundary Value Problems**: Shooting method, finite difference
- **PDEs**: Heat equation, wave equation, Laplace equation
- **Finite Difference Methods**: Forward, backward, central, Crank-Nicolson
- **Stability Analysis**: Von Neumann stability, CFL condition

## Mathematical Topics
- Initial value problems (IVPs)
- Boundary value problems (BVPs)
- Partial differential equations (PDEs)
- Stability and convergence
- Method of lines

## Key Equations
- `dy/dt = f(t, y)` (ODE)
- `∂u/∂t = α∇²u` (Heat equation)
- `∂²u/∂t² = c²∇²u` (Wave equation)
