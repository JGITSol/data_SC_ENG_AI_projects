# Simulated Physics 4: Fluid Dynamics

## Overview
Computational fluid dynamics simulation using lattice Boltzmann method and Navier-Stokes equations for incompressible flow visualization.

## Features
- 2D incompressible fluid flow
- Lattice Boltzmann method (LBM)
- Vorticity visualization
- Flow around obstacles
- Lid-driven cavity flow
- Reynolds number effects

## Physics Concepts
- Navier-Stokes equations
- Continuity equation
- Reynolds number and turbulence
- Boundary conditions
- Vorticity and streamlines

## Key Equations
- `∂u/∂t + (u·∇)u = -∇p/ρ + ν∇²u` (Navier-Stokes)
- `∇·u = 0` (incompressibility)
- `Re = ρuL/μ` (Reynolds number)
