# Applied Mathematics 3: Optimization

## Overview
Implementation of optimization algorithms for unconstrained and constrained problems, including gradient-based and derivative-free methods.

## Features
- **Unconstrained Optimization**: Gradient descent, Newton's method, BFGS
- **Line Search**: Backtracking, Wolfe conditions
- **Constrained Optimization**: Penalty methods, Lagrange multipliers
- **Convex Optimization**: Proximal methods, ADMM
- **Global Optimization**: Simulated annealing, genetic algorithms

## Mathematical Topics
- Convex analysis
- Gradient methods
- Newton-type methods
- Quasi-Newton methods (BFGS, L-BFGS)
- KKT conditions
- Duality theory

## Key Concepts
- Objective function minimization
- Constraint handling
- Convergence criteria
- Second-order methods
