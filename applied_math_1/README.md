# Applied Mathematics 1: Linear Algebra

## Overview
Comprehensive implementation of linear algebra algorithms including matrix decompositions, eigenvalue problems, and linear system solvers.

## Features
- Matrix operations (multiplication, transpose, inverse)
- LU decomposition
- QR decomposition (Gram-Schmidt, Householder)
- Eigenvalue algorithms (power method, QR algorithm)
- Singular Value Decomposition (SVD)
- Linear system solvers (Gaussian elimination, iterative methods)

## Mathematical Topics
- Matrix theory
- Vector spaces
- Eigenvalues and eigenvectors
- Matrix decompositions
- Numerical stability

## Key Algorithms
- **LU Decomposition**: A = LU for solving Ax = b
- **QR Algorithm**: Iterative eigenvalue computation
- **Power Method**: Dominant eigenvalue finding
- **SVD**: A = UΣVᵀ for matrix analysis
